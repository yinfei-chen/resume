# Assets

Put your images in this folder and update the `src` paths in `index.html`.

Suggested files:

- `profile.jpg`: personal headshot
- `project-1.jpg`: project screenshot
- `project-2.jpg`: project screenshot
- `certificate-1.jpg`: certificate or award image

For a resume website, use compressed JPG or WebP images where possible. A profile photo around 800 x 800 px is enough.
